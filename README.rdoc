= HarborFirstTest

Une application très basique développée dans le but de tester comment s'effectue
le déploiement sur AppHarbor et comment y configurer la base de données.


== Technologies

* ASP.NET MVC 3
* .NET Framework 4
* Entity Framework Code First 4.3.1
